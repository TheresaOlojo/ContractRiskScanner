# ContractRiskScanner
AI-powered contract risk scanner (DOCX + PDF)

**AI-powered Contract Risk Scanner**  
Scans DOCX and PDF contracts for risk clauses, highlights keywords, generates CSV/DOCX reports, AI-powered clause summaries, and a visual dashboard.

---

## Features
- Scan DOCX and PDF contracts
- Highlight risky keywords in documents
- AI-powered summaries and safer wording suggestions
- CSV and DOCX report generation
- PDF highlighting for visual review
- Visual dashboard of risk distribution
- Easy-to-edit keyword categories
